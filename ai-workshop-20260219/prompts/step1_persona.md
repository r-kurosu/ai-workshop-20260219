# Step 1: ペルソナ生成

日本人の架空ペルソナを50人生成するスクリプトを作って実行してください。

## 要件

- Azure OpenAI API (gpt-4o-mini) を使用
- 5人ずつ10回に分けて生成
- 各ペルソナにID（P001〜P050）を付与
- 年齢（20〜60代）、職業、家族構成、家計管理スタイル、AI態度にバラつきを持たせる
- 典型的でなく、個人差のあるリアルな人物像にする

## ペルソナ属性に含めるもの

基本属性に加えて、以下の「判断を分岐させる情報」を必ず含めること：

- **家計の逼迫度**：「毎月赤字」「ギリギリ」「余裕あり」「かなり余裕」など
- **月額サブスク支出**：具体的な金額（例：月3,000円、月15,000円）
- **過去のアプリ課金体験**：成功体験や失敗体験（例：「ジム会費を3ヶ月で退会」「Netflixは3年継続」「家計簿アプリを2回挫折」）
- **広告への態度**：「気にならない」「絶対嫌」「多少なら許容」など

## 出力

`output/personas.json` に保存。構造例：

```json
{
  "id": "P001",
  "name": "山田太郎",
  "age": 35,
  "gender": "男性",
  "occupation": "メーカー勤務（営業）",
  "household": "妻と子供2人（5歳、2歳）",
  "annual_income": "600万円",
  "financial_pressure": "ギリギリ。教育費の積立が追いつかない",
  "monthly_subscriptions": "月5,500円（Netflix, Spotify, iCloud）",
  "past_app_experience": "家計簿アプリを2回入れたが1週間で挫折。レシート撮影が面倒だった",
  "ad_tolerance": "無料なら広告は我慢するが、動画広告は嫌",
  "money_management": "妻がExcelで管理。自分はノータッチ",
  "ai_attitude": "便利なら使うが個人情報が心配",
  "background": "共働き。教育費が気になり始めている。新しいアプリを入れるのは面倒なタイプ。"
}
```

生成後、一覧（ID, 名前, 年齢, 職業, 家計逼迫度）をコンソール表示。
